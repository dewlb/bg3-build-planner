# BG3 Companion

A React + TypeScript application for planning Baldur's Gate 3 playthroughs.

## Features

- Item database
- Character builds
- Campaign management
- Gear progression planner

## Tech Stack

- React
- TypeScript
- Vite
- React Query
- React Router
- Tailwind CSS
- Zod

## Future Features

- Character progress tracking
- Shared shopping lists
- Vendor route planner
- Save import support
